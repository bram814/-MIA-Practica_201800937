### Instalacion de Entorno Virtual

```bash 
sudo apt install python3-venv
```
_____

Crear entorno virtual

```bash 
python3 -m venv [directorio]

```
_____

Trabajando en el entorno virtual
_____

```bash 
source [directorio]/bin/activate
```
